# DATAIDEA Science

This project provides structured learning materials covering Python programming, data analysis, statistical methods, machine learning algorithms, and deep learning with PyTorch.

## About This Project

DATAIDEA Science is an open-source educational resource designed to help learners master data science from fundamentals to advanced topics. The project is built using MkDocs and provides interactive Jupyter notebooks, exercises, and comprehensive documentation for self-paced learning.

## Topics Covered

### Python Programming
- **Basics**: Variables, numbers, strings, operators, containers, flow control, modules, file handling
- **Built-in Functions**: Filter, map, reduce functions
- **Libraries**: NumPy, Pandas, Matplotlib crash courses
- **Exercises**: Basics exercises, objective quizzes, true/false quizzes, end-of-course exercises

### Data Collection and Visualization
- NumPy and Pandas for data manipulation
- Matplotlib for data visualization
- Data exploration and cleaning techniques
- Handling missing data
- Real-world data analysis exercises (weather data, Nobel Prize data)

### Machine Learning
- **Overview**: Introduction to machine learning concepts
- **Training Models**: Understanding model training and evaluation
- **Unsupervised Learning**: Clustering and dimensionality reduction techniques
- **Feature Engineering**: Feature selection, scaling, normalization, and standardization
- **Model Evaluation**: Classification and regression metrics
- **Advanced Techniques**: Scikit-learn pipelines, GridSearchCV for hyperparameter tuning
- **Exercises**: Quizzes on classification metrics, missing data handling, and pipelining

### Deep Learning
- **Tensors & Datasets**: 1D/2D tensors, derivatives, graphs, simple and pre-built datasets
- **Linear Regression**: 1D regression, one parameter, slope & bias training
- **Linear Regression in PyTorch**: SGD, mini-batch gradient descent, PyTorch implementation, training & validation
- **Multiple Input Linear Regression**: Multiple features, multi-target regression
- **Logistic Regression**: Prediction, cross-entropy loss, softmax activation
- **Practice**: Custom datasets, DataLoaders, transforms
- **Shallow Networks**: Hidden layers, multiple neurons, XOR problem, MNIST classification, activation functions
- **Deep Networks**: Multi-layer networks, dropout, initialization techniques (Xavier, He), momentum, batch normalization
- **CNNs**: Convolution basics, activation & pooling, multiple channels, simple CNN implementations

### Time Series Analysis
- Introduction to time series concepts
- Time series analysis techniques
- Forecasting methods
- Python implementation examples

### Mathematics & Statistics
- Overview of mathematics and statistics for data science
- Eigenvalues and eigenvectors
- Descriptive statistics
- Inferential statistics
- Implementing statistical models
- Hypothesis testing

### Extras
- Customer analysis case studies
- How KNN (K-Nearest Neighbors) works
- Handling imbalanced data
- Classification metrics deep dive

## Getting Started

The project uses MkDocs for documentation. To build and serve the documentation locally:

```bash
# Install dependencies
pip install -r requirements.txt

# Serve the documentation
mkdocs serve
```

Visit the live site at [https://science.dataidea.org/](https://science.dataidea.org/)
